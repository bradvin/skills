# CTR Benchmarks by Position

## Desktop organic CTR (industry average, informational content)

| Position | CTR range  | Use for gap analysis |
|----------|-----------|----------------------|
| 1        | 25–32%    | Benchmark: 28%       |
| 2        | 13–17%    | Benchmark: 15%       |
| 3        | 9–12%     | Benchmark: 11%       |
| 4        | 6–9%      | Benchmark: 7%        |
| 5        | 5–7%      | Benchmark: 6%        |
| 6        | 3–5%      | Benchmark: 4%        |
| 7        | 3–4%      | Benchmark: 3.5%      |
| 8        | 2–4%      | Benchmark: 3%        |
| 9        | 2–3%      | Benchmark: 2.5%      |
| 10       | 2–3%      | Benchmark: 2%        |
| 11–15    | 1–2%      | Benchmark: 1.5%      |
| 16–20    | 0.5–1.5%  | Benchmark: 1%        |

Source: Aggregate data from multiple SEO studies (Backlinko, FirstPageSage, Advanced Web Ranking).
Use as directional benchmarks, not absolutes — CTR varies significantly by:
- Query type (branded vs generic)
- Presence of SERP features (AI Overview, featured snippet, PAA boxes push organic down)
- Device (mobile CTR is lower across positions)
- Niche competitiveness

## SERP feature adjustments

When SERP features are present, organic CTR at all positions drops:
- **AI Overview present**: reduce position 1–3 benchmarks by ~15–25%
- **Featured snippet present**: position 1 CTR may drop 5–10% (snippet takes clicks)
- **PAA box above organic**: reduce position 1–5 by ~10%
- **Ads occupying top 3 spots**: reduce organic position 1 by ~15–20%

## CTR gap calculation

```
CTR gap % = (expected_CTR - actual_CTR) / expected_CTR × 100
```

Example: position 10, expected 2%, actual 0.11% → gap = (2 - 0.11) / 2 × 100 = **94.5% below expected**

## Monthly click opportunity estimate

```
Additional clicks/month = impressions × (target_CTR - actual_CTR)
```

Conservative target: 1.5× expected CTR (achievable with a strong title)
Aggressive target: 2× expected CTR (top-performing titles in competitive niches)

Example: 31,432 impressions, actual 0.11%, target 2%
→ Additional clicks = 31,432 × (0.02 - 0.0011) = ~594 additional clicks/month
